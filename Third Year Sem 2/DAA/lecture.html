Lecture 9

0/1 KnapSack using DP

k(i,c) = max[(Pi + k(i - 1, c - wi),k(i - 1, c))]

k(i,c) = 0 [i == 0 || c == 0]

k(i,c) = k(i,c) [wi > c]
